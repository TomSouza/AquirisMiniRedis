# AquirisMiniRedis
Projeto teste para a vaga de desenvolvedor backend na Aquiris

O Objetivo do projeto é simular 10 comandos do Redis utilizando dados em memoria de maneira atômica.

Os comandos são: 
 - Set **|** Set Ex **|** Get **|** Del **|** DbSize **|** Incr **|** ZAdd **|** ZCard **|** ZRank **|** ZRange
